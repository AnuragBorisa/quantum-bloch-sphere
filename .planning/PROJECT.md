# Bloch Sphere Visualizer

## Goal
Build a single-page 3D web app that visualizes a single-qubit state on the Bloch sphere. Users can pick preset states or enter custom angles, apply gates, and see real-time rotation with numeric readouts.

## Scope (v1)
- 3D Bloch sphere with axes and state vector.
- Preset states and custom (theta, phi) input.
- Gate controls: RX, RY, RZ + X, Y, Z.
- Immediate visual feedback.

## Out of Scope (v1)
- Coin visualization.
- Multi-qubit circuits.
- Backend or persistence.
